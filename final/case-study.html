Project Overview

Cookbook Corner is a fully responsive and user-friendly web application that integrates HTML, CSS, PHP, and MySQL to create a seamless recipe browsing experience. The project features a custom-built database paired with a dynamic front end, allowing users to search for and filter recipes by cuisine, diet, and keywords. Developed as part of the IDM 232 Web Scripting II course under Professor Phil Sinatra, this 10-week intensive project highlights my technical skills in server-side scripting and responsive design while delivering a platform that meets both user needs and assignment objectives.

Context & Challenge

Background

This project was developed for my IDM 232 Web Scripting II course, focusing on learning the fundamentals of server-side scripting. My task was to design and develop a unique online recipe website.

Process & Insights

Design Stage

To establish the site’s professional yet warm and comforting aesthetic, I began by brainstorming branding ideas. This resulted in the website’s name, Cookbook Corner, accompanied by a logo featuring a recipe book.

After establishing the brand and feel, I created three design cards that showcased different combinations of font styles and color palettes. These options were presented to users for feedback, and the third card emerged as the clear favorite, providing the foundation for the website’s aesthetic direction. From this feedback, I developed a comprehensive style guide that established a cohesive set of colors and fonts, ensuring visual harmony throughout the site.

With the branding solidified, I moved on to creating a high-fidelity wireframe to map out the website’s layout and navigation. Peer feedback on the wireframe was important in refining the user flow, ensuring the design supported seamless and intuitive navigation.

In addition to ensuring usability, I wanted to add unique elements to make the website more engaging. To achieve this, I designed a homepage featuring a "Pick of the Day" section that randomizes a recipe daily (for demonstration purposes, it randomizes with every refresh), along with a "Latest Recipes" section to showcase newly added options. I also introduced a "Surprise!" button, allowing users to generate a random recipe when they’re unsure what to cook, adding a fun and interactive feature to the experience.

Development Stage

The development process began with creating the structure and layout of the website using HTML and CSS. To ensure consistency and efficiency, I used one recipe as a template to plan the structure of the pages to make sure that it was functioning as intended for responsive design. This initial stage served as a solid foundation for the project.

With the front end established, the second phase of development focused on database integration and backend functionality. I set up a local server and MySQL database using MAMP to store the recipes and their attributes, including cuisine, protein, diet, and other keywords. Using PHP, I wrote scripts to dynamically fetch data from the database and insert it into the HTML templates. This allowed the website to transition from a static to a dynamic application, pulling live data for display.

The final phase involved implementing the search and filtering functionality. I designed the interface to allow users to search by keywords and apply filters for cuisine, protein, and diet. On the backend, I utilized PHP to handle the queries and return relevant results dynamically. This process required significant troubleshooting and optimization to ensure smooth and accurate filtering across multiple criteria.

The Solution

Cookbook Corner is a cohesive and dynamic platform that combines back-end functionality with a responsive, visually appealing front-end. Users can browse recipes, search by keywords, and filter based on multiple preferences. The site also adapts to various screen sizes, ensuring accessibility across devices. Its cozy branding and engaging interface invite users to explore recipes effortlessly.

The Results

The final version of Cookbook Corner featured seamless PHP and MySQL integration for dynamic content management and a responsive design for use on both desktop and mobile devices. Its filtering system allows users to efficiently search and narrow results by keywords, cuisine, protein, and diet.
